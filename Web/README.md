# Want To Try | เด็กอยากลอง

🗳️ **เว็บไซต์หาเสียงพรรคสภานักเรียน "Want To Try | เด็กอยากลอง"**

## 📁 โครงสร้างโปรเจกต์

```
├── index.html          # หน้าแรก
├── about.html          # เกี่ยวกับพรรค
├── candidates.html     # ผู้สมัคร
├── policies.html       # นโยบาย
├── news.html           # ข่าวสาร
├── contact.html        # ติดต่อ
├── ai-chat.html        # ระบบ AI Chat
├── results.html        # ผลคะแนน
├── submissions.html    # ข้อเสนอแนะ
├── style.css           # สไตล์หลัก
├── script.js           # JavaScript หลัก
├── images/             # รูปภาพทั้งหมด
│   ├── Logo.png
│   └── candidates/     # รูปผู้สมัคร
└── ระบบนับคะแนน/        # ระบบนับคะแนนเลือกตั้ง
```

## 🚀 การใช้งาน

### วิธีที่ 1: เปิดไฟล์โดยตรง
เปิดไฟล์ `index.html` ด้วยเว็บเบราว์เซอร์

### วิธีที่ 2: GitHub Pages
1. Push โค้ดไปยัง GitHub Repository
2. ไปที่ Settings → Pages
3. เลือก Source เป็น `main` branch
4. เว็บไซต์จะถูก deploy โดยอัตโนมัติ

## ✨ Features

- 📱 **Responsive Design** - รองรับทุกอุปกรณ์
- 🤖 **AI Chat Assistant** - ถามคำถามเกี่ยวกับนโยบายได้
- ⏱️ **Countdown Timer** - นับถอยหลังสู่วันเลือกตั้ง
- 🎨 **Modern UI** - ดีไซน์ทันสมัย สวยงาม

## 🛠️ Technologies Used

- HTML5
- CSS3 (Vanilla CSS)
- JavaScript
- Font Awesome Icons
- Google Fonts (Prompt)

## 📝 หมายเหตุ

- Form ติดต่อต้องตั้งค่า Google Apps Script URL ใน `script.js`
- AI Chat ใช้ระบบ Knowledge Base ในตัว (ไม่ต้อง API key)
